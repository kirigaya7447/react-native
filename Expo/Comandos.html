Aqui estão os comandos para iniciar um projeto com o expo:

  1. npx create-expo-app --template*
  2. npx expo init | npm init
  3. npx expo start | npm start

Ao escolher o template prefira o Blank

Arrumar dependências no projeto:

  1. npm install expo@latest
  2. npm audit fix --force
  3. npx expo install --fix
  4. npx expo install --check
  5. npx expo-doctor
